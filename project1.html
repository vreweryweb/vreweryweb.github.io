DATA

VISUALISATION

Data is powerful only when it can get you relevant insights. The scale of data-sets is enormous in contemporary times. Analyzing and acting upon it takes a lot of intensive effort, specially more on the limited flat screens which are apt for visualizing large data sets. Virtual reality provides 360 degrees of space to visualize, analyze and make sense of all this data.

Scroll Down


Reading, analysing, interpreting, visualizing and gaining insights are key aspects of Data Analytics that are currently done using 2D display mediums. With increasing large amounts of data inflow for analysis (Big data) this task is becoming more intensive further limited by the two dimensional screens. This experiment done in collaboration with MetarvrseAR/VR tech start-up, explores the possiblity of using virtual reality as a medium for making sense of data.
